Detailed Description

Main helper for AFS search reply.

This helper is intended to be initiliazed with the reply provided by AfsSearchQueryManager::send. It allows to manage replies of the first replyset, including facets and pager. Connection and query errors are managed in a uniform way to simplify integration.

Examples:
full_example.php.

Parameters
$response[in] result from AfsSearchQueryManager::send call.
$facet_mgr[in] AfsFacetManager used to create appropriate queries.
$query[in] query which has produced current reply.
$coder[in] AfsQueryCoderInterface if set it will be used to create links (default: null).
$format[in] if set to AFS_ARRAY_FORMAT (default), all underlying helpers will be formatted as array of data, otherwise they are kept as is. See Helper format for more details.
$visitor[in] text visitor implementing AfsTextVisitorInterface used to extract title and abstract contents. If not set, default visitor is used (see AfsReplyHelper).
Exceptions
InvalidArgumentExceptionwhen one of the parameters is invalid.
