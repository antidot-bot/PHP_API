Detailed Description

Interface for encoding/decoding queries.

Encoded query is used to generate parameters suitable to build link for specific action.
Query is decoded from array of parameters. This array can come from $_GET value.

For example:

  • Suppose you are looking for shoes.
  • First you search for the word shoes. The URL can be of the form: http://name.some.domain/foo?query=shoes.
  • Then some results are presented along with size and color facets.
  • You have to build the query object from URL parameters, otherwise you will get red shoes but also all red items. You can initialize your query with: $query = $query_coder->build_query($_GET);.
  • Now you want to filter on red color, you have to add a filter to the query: $query = $query->add_filter('color', 'red');
  • Then you can generate parameters suitable for a valid HTTP link by calling $params = $query_coder->generate_parameters($query);.
