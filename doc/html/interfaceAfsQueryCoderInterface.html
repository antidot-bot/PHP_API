Detailed Description

Interface for encoding/decoding queries.

Encoded query is used to generate parameters suitable to build link for specific action.
Query is decoded from array of parameters. This array usually comes from $_GET value.

For example:

  • Suppose you are looking for shoes in your favorite web store.
  • First you search for the word shoes.
  • Then some results are presented along with size and color facets.
  • Now you want to filter on red color.
  • You have to build the query object from URL parameters, otherwise if you filter on the color only, you will get all red items. So you should initialize your query with: 
     $query = $query_coder->build_query($_GET);
  • Now you want to filter on red color, you have to add a filter to the query: 
     $query = $query->add_filter('color', 'red');
  • Then you can generate parameters suitable for a valid HTTP link by calling: 
     $params = $query_coder->generate_parameters($query);
    .
