$unusedHum...

Exceptions

Example with name=null:
Input JSON client data: 
    {
        "clientData": [
            {
                "contents": { "data": [ "afs:t": "KwicString", "text": "some text" ] },
                "id": "data1",
                "mimeType": "application/json"
            }
        ]
    }
Call to get_text(null) will return 
{"data":["afs:t":"KwicString","text":"some text"]}
.
