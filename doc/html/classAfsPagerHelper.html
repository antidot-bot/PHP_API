Detailed Description

Helper for pager.

This class allows to manage:

  • previous page if it exists.
  • next page if it exists.
  • numbered pages.

format (  ) 

Retrieve pages as array.

All data are store in key => value format:

  • <page number>: query or link associated to this page number,
  • next: query or link for the next page (if it exists),
  • previous: query or link for the previous page (if it exists),
  • current: current page number.
Returns:
array filled with key and values.
