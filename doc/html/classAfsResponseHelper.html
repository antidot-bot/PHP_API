Detailed Description

Main helper for AFS search reply.

This helper is intended to be initiliazed with the reply provided by AfsSearchQueryManager::send. It allows to manage replies of the first replyset, including facets and pager. Connection and query errors are managed in a uniform way to simplify integration.

Examples:

full_example.php.
