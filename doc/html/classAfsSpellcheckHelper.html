Detailed Description

Spellcheck helper.

Spellcheck helper allows to get access to main spellcheck data:

  • raw text: it can be used to generate new query or present result without specific formatting.
  • formatted text: it is used to present formatted spellcheck result.
  • query: new query to submit to AFS search engine with spellcheck result as the new words query.
  • link: the link to put in HTML result page to allow query of AFS serach engine. This link is generated only when a query coder has been provided to spellcheck manager.
