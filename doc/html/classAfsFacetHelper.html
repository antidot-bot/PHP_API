Parameters
$facet[in] root facet element.
$facet_mgr[in] AfsFacetManager with properly configured facets.
$query[in] AfsQuery which has produced current reply.
$coder[in] AfsQueryCoderInterface if set it will be used to create links (default: null).
$format[in] if set to AFS_ARRAY_FORMAT (default), format facet and its values as array, otherwise, helpers are returned as is.
