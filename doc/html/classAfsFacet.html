Construct new facet with specified parameters.

Parameters
$id[in] facet id defined in feed.xml on indexation side.
$type[in] facet type defined in feed.xml on indexation side (see Facet type for available types).
$mode[in] facet mode (see Facet mode):
  • replace (default): when new value of the facet is selected, it replaces previous one,
  • add : when new value of the facet is selected, it is combined with previous ones.
$combination[in] 'and' or 'or' (default) combination used to combine multiple facet values (see Facet value combination).
$sticky[in] defines the stickyness of the facet (see Facet stickyness). Sticky combination is defaultly set when this parameter is not provided and facet combination is set to AFS_FACET_OR.
Exceptions
InvalidArgumentExceptioninvalid parameter value provided for mode, combination or sticky parameter.
